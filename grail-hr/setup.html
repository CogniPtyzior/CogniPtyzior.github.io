Installation locale

Un environnement local pensé pour WSL, Docker Desktop et DDEV.

Le dépôt se lance comme un projet WordPress DDEV avec un frontend Nuxt indépendant.

URLs locales

WordPress admin

https://grail-hr.ddev.site/wp-admin

API REST

https://grail-hr.ddev.site/wp-json/grail-hr/v1

Nuxt

http://localhost:3000

Démarrage

cd /mnt/c/Users/<user>/Desktop/Dev/Grail-HR
ddev start
ddev launch /wp-admin
cd wp-content/plugins/grail-hr/frontend/nuxt
npm install
npm run dev
Pré-requis recommandés

Windows 11, WSL2/Ubuntu, Docker Desktop avec intégration WSL, DDEV installé côté Ubuntu, Composer et Node.js via nvm.

Prototype fonctionnel développé comme démonstration produit et technique. Données fictives uniquement.
