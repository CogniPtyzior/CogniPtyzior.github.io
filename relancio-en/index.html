What Relancio does

Relancio retrieves emails, prepares a reply with local AI, then lets you review, edit, save or send it. The user always remains in control of sending.

Local-first

Emails, drafts, replies, prompts, business rules and AI data stay on the user’s computer. Optional remote access is used to reach this computer, not to host your data.

Mobile as a complement

The web/mobile portal is a convenient option for validating replies remotely. It requires private access provisioned by the author, which may or may not be provided.

Install Relancio

Relancio is a portable application: it does not require a traditional installation. Simply download the ZIP archive, extract it, then launch the application by double-clicking Relancio.App.exe.

Installation steps

  1. Download the Relancio ZIP file.
  2. Right-click the downloaded ZIP file.
  3. Click Properties.
  4. At the bottom of the window, if the option is available, check Unblock.
  5. Click Apply, then OK.
  6. Extract the ZIP file.
  7. Open the extracted folder, then double-click Relancio.App.exe.

Why can Windows display a warning?

Since Relancio is downloaded from the Internet and runs directly without an installer, Windows may sometimes display a security message the first time it is launched. This is common behavior for recent portable applications.

Unblocking the ZIP file before extracting it prevents Windows from passing this security mark on to the extracted files.
