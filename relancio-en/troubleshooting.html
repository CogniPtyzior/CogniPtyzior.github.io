Troubleshooting and glossary

Solve common issues and understand technical terms.

Common issues

The AI indicator is not green

Check that Ollama is available, that the local address is correct, that the model is installed and that the AI assistant test works. If the local AI assistant is not installed yet, use the automatic installation offered in the Local AI tab.

The SMTP indicator is not green

Check server, port, security, username, password and any required app password. Launch a test email from the Sending tab.

No email appears

Check the POP3/IMAP configuration, retrieval frequency, the Retrieve button and the technical account credentials.

Web/mobile access does not work

Check that access has been provisioned, that the web/mobile password is defined, that the private address is correct and that the tunnel is active when remote access is expected.

Gmail refuses the connection

Check the Google account POP/IMAP settings and use an app password when Gmail requires it.

Quick glossary

POP3
Reception mode that retrieves messages from a mailbox.
IMAP
Reception mode more oriented toward synchronization with the remote mailbox.
SMTP
Protocol used to send emails.
Ollama
Local AI server used to run generative models on the computer.
Provisioning
Association of a Relancio workstation with a private remote access address, through a code provided by the author.
Business kit
Set of rules and prompts adapted to a professional context.

Appearance

Relancio provides a theme choice in the Appearance tab. The documentation uses the application’s dark blue style for consistency.
