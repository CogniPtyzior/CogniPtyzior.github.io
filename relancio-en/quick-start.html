Quick start

Recommended steps to configure Relancio in the right order.

Recommended path

  1. General: check the domain used by Relancio, then enter the company name, the sender name displayed in emails and the signature.
  2. Reception: configure the POP3 or IMAP account that receives client requests.
  3. Sending: configure the SMTP server and send a test email.
  4. Local AI: check Ollama, install the assistant if needed, choose the model and test the AI.
  5. Business kits: import a ZIP kit if you want to adapt replies to a business context.
  6. Processing: retrieve emails, review the proposed reply, save it or send it.

Remember

No email is sent automatically. The Send button always corresponds to an explicit user action.

Status indicators

The Local, Tunnel, SMTP and AI indicators give a quick overview of useful services. They help with diagnosis without replacing the settings pages.

Dedicated mailbox recommended

For better security and clarity, ideally configure a technical mailbox dedicated to Relancio.
