Installer Relancio

Relancio est une application portable : elle ne nécessite pas d’installation classique. Il suffit de télécharger l’archive ZIP, de la décompresser, puis de lancer l’application en double-cliquant sur Relancio.App.exe.

Étapes d’installation

  1. Téléchargez le fichier ZIP de Relancio.
  2. Faites un clic droit sur le fichier ZIP téléchargé.
  3. Cliquez sur Propriétés.
  4. En bas de la fenêtre, si l’option est présente, cochez Débloquer.
  5. Cliquez sur Appliquer, puis sur OK.
  6. Décompressez ensuite le fichier ZIP.
  7. Ouvrez le dossier extrait, puis double-cliquez sur Relancio.App.exe.

Pourquoi Windows peut afficher un avertissement ?

Comme Relancio est téléchargé depuis Internet et se lance directement sans installateur, Windows peut parfois afficher un message de sécurité au premier lancement. C’est un comportement courant pour les applications portables récentes.

Débloquer le fichier ZIP avant de le décompresser évite que Windows applique ce marquage de sécurité aux fichiers extraits.
