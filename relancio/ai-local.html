Relancio

IA locale

Comprendre Ollama, les modèles locaux, le timeout et la régénération.

Comprendre l’IA locale

Relancio s'appuie sur Ollama pour interroger un modèle installé sur le poste de l'utilisateur. L'adresse locale typique est http://localhost:11434.

Le modèle reçoit les informations utiles du mail et les règles métier locales afin de produire une proposition de réponse. Cette proposition reste modifiable et doit être validée par l'utilisateur.

Installation de l’assistance IA

Si l'assistance IA locale n'est pas encore installée, Relancio propose une installation automatique. Si cette installation ne fonctionne pas sur le poste, vous pouvez télécharger Ollama depuis le site officiel, puis revenir dans Relancio pour actualiser les modèles.

Timeout et fallback

Le timeout définit combien de temps Relancio attend la réponse du modèle. Si Ollama ne répond pas ou dépasse le délai, Relancio n’envoie rien automatiquement : vous pouvez corriger manuellement, sauvegarder ou relancer une génération.

Régénérer

Le bouton Régénérer redemande une proposition à l’IA locale pour le mail sélectionné. Il est utile après modification du contexte ou si la première réponse n’est pas satisfaisante.

Responsabilité de validation

Une IA générative peut produire une réponse inexacte, incomplète ou mal adaptée. Relisez toujours le contenu, l’objet et le destinataire avant d’envoyer.
