Developer workflow

Workflow

Recommended reading flow

  1. Authenticate with a demo or admin account.
  2. Create supporting data: company, customer, tax rate and product.
  3. Create a draft invoice from validated request data.
  4. Issue it to lock invoice details and assign an invoice reference.
  5. Register payments and read the invoice projection.
