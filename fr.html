Prototype IA RH · Démo publique

Grail HR

WordPress / Nuxt

Grail HR est un prototype fonctionnel qui transforme des CV PDF en profils candidats structurés. Il démontre un workflow RH assisté par IA avec import temporaire de CV, analyse structurée, relecture humaine, validation, administration et exports.

Valeur produit

  • Workflow : import PDF, analyse IA structurée, relecture éditable et validation humaine
  • Backend : plugin WordPress avec API REST, gestion des accès, logs et exports
  • Frontend : application Nuxt indépendante pour créer, rechercher et relire les profils
  • Limites : données fictives, pas de décision automatisée, pas de matching d’offres et pas d’usage production sans audit

Assistant email IA local-first · Release desktop

Relancio

Relancio est un produit desktop local-first qui aide à traiter les mails entrants, générer des brouillons de réponse assistés par IA, les relire, les corriger et les envoyer manuellement. Il met l’accent sur le contrôle utilisateur, la confidentialité et les workflows email concrets, avec IA locale, kits métiers et accès web/mobile optionnel.

Valeur produit

  • Desktop : .NET, Avalonia UI
  • IA : support LLM local avec Ollama
  • Messagerie : réception POP3/IMAP, envoi SMTP et validation manuelle
  • Distribution : build Windows principal, build Linux expérimental, kits métiers et documentation utilisateur

Windows est la plateforme principale supportée. Le build Linux est fourni en aperçu, testé rapidement, et peut présenter quelques fonctionnalités en moins en raison de contraintes propres à Linux.

Produit desktop · Release portable

Dynamic Grep

Windows / Linux

Application desktop local-first pour la recherche, l’extraction, l’inspection et le reporting sur fichiers locaux. Dynamic Grep est conçu comme un outil d’analyse en lecture seule permettant de trouver, comprendre, comparer et exporter des résultats sans modifier les fichiers d’origine.

Stack technique

  • Focus : local-first, confidentialité, rapports, actions fichiers sûres et documentation hors ligne

Qualification de contact B2B

Grail Signal

Grail Signal est un projet de démonstration PHP pour qualifier un contact B2B probable à partir d’entrées entreprises. Il fusionne des signaux métier mockés, applique un score de confiance, conserve la provenance des champs et route les preuves faibles ou conflictuelles vers une revue humaine.

Stack technique

  • Backend : PHP, Composer, PHPUnit
  • Architecture : structure hexagonale légère avec Domain, Application, Ports et Infrastructure
  • Workflow : entrée CSV, sources mockées, fusion d’évidence, scoring et sortie JSON/JSONL
  • Focus : traçabilité, règles de suppression, filtrage des emails personnels et revue humaine
