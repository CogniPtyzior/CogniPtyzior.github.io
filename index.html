AI HR prototype · Public demo

Grail HR

WordPress / Nuxt

Grail HR is a functional prototype that turns PDF resumes into structured candidate profiles. It demonstrates an AI-assisted HR workflow with temporary resume import, structured analysis, human review, validation, administration and exports.

Product value

  • Workflow: PDF import, structured AI analysis, editable review and human validation
  • Backend: WordPress plugin with REST API, access management, logs and exports
  • Frontend: independent Nuxt application for profile creation, search and review
  • Boundaries: fictional data, no automated decision-making, no job matching and no production use without audit

Local-first AI email assistant · Desktop release

Relancio

Relancio is a local-first desktop product that helps users process incoming emails, generate AI-assisted reply drafts, review them and send them manually. It focuses on control, privacy and practical email workflows, with local LLM support, business kits and optional web/mobile access.

Product value

  • Desktop: .NET, Avalonia UI
  • AI: local LLM support with Ollama
  • Mail workflow: POP3/IMAP reception, SMTP sending and manual validation
  • Distribution: Windows primary build, experimental Linux build, business kits and user documentation

Windows is the primary supported platform. The Linux build is provided as a lightly tested preview and may miss a few features because of Linux-specific constraints.

B2B contact qualification batch

Grail Signal

Grail Signal is a PHP demonstration project for qualifying a likely B2B payment contact from company inputs. It merges local mocked business signals, applies confidence scoring, keeps field-level provenance and routes weak or conflicting evidence to human review.

Technical stack

  • Backend: PHP, Composer, PHPUnit
  • Architecture: lightweight hexagonal structure with Domain, Application, Ports and Infrastructure
  • Workflow: CSV input, mocked source lookup, evidence merge, scoring and JSON/JSONL output
  • Focus: traceability, suppression rules, personal email filtering and human-review gating
