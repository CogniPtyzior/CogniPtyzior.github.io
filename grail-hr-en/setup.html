Local setup

A local environment built for WSL, Docker Desktop and DDEV.

The repository runs as a DDEV WordPress project with an independent Nuxt frontend.

Local URLs

WordPress admin

https://grail-hr.ddev.site/wp-admin

REST API

https://grail-hr.ddev.site/wp-json/grail-hr/v1

Nuxt

http://localhost:3000

Start the project

cd /mnt/c/Users/<user>/Desktop/Dev/Grail-HR
ddev start
ddev launch /wp-admin
cd wp-content/plugins/grail-hr/frontend/nuxt
npm install
npm run dev
Recommended prerequisites

Windows 11, WSL2/Ubuntu, Docker Desktop with WSL integration, DDEV installed in Ubuntu, Composer and Node.js through nvm.

Functional prototype built as a product and technical demo. Fictional data only.
