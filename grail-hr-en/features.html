Features

A complete flow to turn a resume into a usable profile.

The prototype covers the core cycle: import, analyze, review, correct, validate, archive and export.

1

Secure resume import

PDF only, 5 MB maximum. The source resume and extracted raw text are not kept after analysis.

2

Manual creation

A profile can be created without a resume, then enriched manually by the team.

3

OpenAI analysis

The backend calls OpenAI with a versioned prompt and schema to receive structured output.

4

Editable review

Extracted information remains editable before validation.

5

Search and filters

The profile list combines global search, status, seniority, sorting and server-side pagination.

6

Exports and logs

The WordPress admin provides JSON/CSV exports, secured logs and technical diagnostics.

Functional prototype built as a product and technical demo. Fictional data only.
